Automation Exercise Test Automation Framework

Overview

This project is a Test Automation Framework developed for the Automation Exercise website. It covers both UI and API testing using modern automation practices and design patterns to ensure maintainability, scalability, and reusability.

Technology Stack
Java
Selenium WebDriver
TestNG
REST Assured
Maven
Design Patterns

The framework is designed using the following design patterns:

Page Object Model (POM)
Fluent Pattern
Bot Pattern
Factory Pattern
